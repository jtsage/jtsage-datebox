Flip Default

Using the slide mode is as simple as setting the 'mode' option to "flipbox".

<label for="mydate">Some Date</label>

<input name="mydate" id="mydate" type="date" data-role="datebox"
   data-options='{"mode": "defflip"}'>

Flip Time Mode Default

Using the slide mode is as simple as setting the 'mode' option to "timeflipbox".

<label for="mydate">Some Date</label>

<input name="mydate" id="mydate" type="date" data-role="datebox"
   data-options='{"mode": "deftimeflip"}'>

Flip Time Mode 12 Hour Clock

Using the slide mode is as simple as setting the 'mode' option to "timeflipbox".

<label for="mydate">Some Date</label>

<input name="mydate" id="mydate" type="date" data-role="datebox"
   data-options='{"mode": "deftimeflip", "timeFormat": 12}'>
