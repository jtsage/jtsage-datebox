Available Options

  • mode

    Purpose:Mode of operation
    Default: 'datebox'
    Options: 'datebox', 'slidebox', 'calbox', 'timebox', 'durationbox', 'flipbox', 'timeflipbox'

  • Date Formats

    • Output Format

    • dateFormat

      DEPRECIATED/CHANGED: Part of the i8n Configuration nowPurpose:Format the returned date (Note: this is an override of the i8n provided format)
      Default: 'YYYY-MM-DD' for dates
      Modes: all date 

      	YYYY: 4 Digit Year
	MM : 2 Digit Month (zero pad)
	mm : 1 or 2 Digit Month
	mmm : Text Month (monthsOfYear)
	mmmm : Text Month, abbreviated (monthsOfYearShort)
	DD : 2 Digit Day (zero pad)
	dd : 1 or 2 Digit Day
	ddd : Day of week (daysOfWeek)
	dddd : Day of week, abbreviated (daysOfWeekShort)
	SS : Ordinal Suffix (4th, 3rd, 1st, etc)
	HH : 2 Digit Hour, 24 Hour Clock (zero pad)
	GG : 1 or 2 Digit Hour, 24 Hour Clock
	hh : 2 Digit Hour, 12 Hour Clock (zero pad)
	gg : 1 or 2 Digit Hour, 12 Hour Clock
	ii : 2 Digit Minutes (zero pad)
	ss : 2 Digit Seconds (zero pad)
	aa : Lower case meridiem letters
	AA : Upper case meridiem letters

    • headerFormat

      DEPRECIATED: Part of the i8n Configuration now
      Purpose:Format the date header
      Default: 'ddd, mmm dd, YYYY'
      Modes: android, slide 

      	YYYY: 4 Digit Year
	MM : 2 Digit Month (zero pad)
	mm : 1 or 2 Digit Month
	mmm : Text Month (monthsOfYear)
	mmmm : Text Month, abbreviated (monthsOfYearShort)
	DD : 2 Digit Day (zero pad)
	dd : 1 or 2 Digit Day
	ddd : Day of week (daysOfWeek)
	dddd : Day of week, abbreviated (daysOfWeekShort)
	SS : Ordinal Suffix (4th, 3rd, 1st, etc)
	HH : 2 Digit Hour, 24 Hour Clock (zero pad)
	GG : 1 or 2 Digit Hour, 24 Hour Clock
	hh : 2 Digit Hour, 12 Hour Clock (zero pad)
	gg : 1 or 2 Digit Hour, 12 Hour Clock
	ii : 2 Digit Minutes (zero pad)
	ss : 2 Digit Seconds (zero pad)
	aa : Lower case meridiem letters
	AA : Upper case meridiem letters

    • timeOutput

      DEPRECIATED: Part of the i8n Configuration nowPurpose:Format the returned time
      Default: 'GG:ii AA' or 'HH:ii' dependant on timeFormat
      Modes: timebox, timeflipbox 

      	YYYY: 4 Digit Year
	MM : 2 Digit Month (zero pad)
	mm : 1 or 2 Digit Month
	mmm : Text Month (monthsOfYear)
	mmmm : Text Month, abbreviated (monthsOfYearShort)
	DD : 2 Digit Day (zero pad)
	dd : 1 or 2 Digit Day
	ddd : Day of week (daysOfWeek)
	dddd : Day of week, abbreviated (daysOfWeekShort)
	SS : Ordinal Suffix (4th, 3rd, 1st, etc)
	HH : 2 Digit Hour, 24 Hour Clock (zero pad)
	GG : 1 or 2 Digit Hour, 24 Hour Clock
	hh : 2 Digit Hour, 12 Hour Clock (zero pad)
	gg : 1 or 2 Digit Hour, 12 Hour Clock
	ii : 2 Digit Minutes (zero pad)
	ss : 2 Digit Seconds (zero pad)
	aa : Lower case meridiem letters
	AA : Upper case meridiem letters

    • durationFormat

      Purpose:Format the returned duration
      Default: 'DD ddd, hh:ii:ss'
      Modes: durationbox 

      	DD: Number of Days (no padding)
	ddd : Text "Day" or "Days" as appropriate from durationDays option
	hh : 2 or more Digit Hours (zero pad) - can exceed 24 if days not used
	ii : 2 or more Digit Minutes (zero pad) - can exceed 59 if hours not used
	ss : 2 or more Digit Seconds (zero pad) - can exceed 59 if minutes not used

    • durationShort

      OPTION REMOVED

    • durationNoDays

      OPTION REMOVED

    • Input Format

    • timeFormat

      DEPRECIATED: Part of the i8n Configuration nowPurpose:Set either a 12 or 24 hour clock
      Default: 24
      Modes: time

    • fieldsOrder

      Purpose:Define the order of the fields (as an override. Otherwise, from the i8n settings)
      Default: android: ['m', 'd', 'y'], slide: ['y', 'm', 'd'], time: ['h', 'i', 'a'] ('a' only displays on 12 hour format)
      Modes: android, time, slide

      	y : Year
	m : Month
	d : Day of Month (date)
	h : Hour
	i : Minute
	a : Meridiem

    • durationOrder

      Purpose:Define the order of the fields
      Default: ['d', 'h', 'i', 's']
      Modes: duration

      	d : Days
	h : Hours
	i : Minutes
	s : Seconds

    • calShowDays

      Purpose:Boolean show day names in grid
      Default: true
      Modes: calendar

    • calShowOnlyMonth

      Purpose:Show *only* this month, do not fill in empty boxes
      Default: false
      Modes: calendar

    • calStartDay

      Purpose:Day of week to start calendar on (numeric - 0=Sun, 1=Mon,... 6=Sat)
      Default: 0
      Modes: calendar

    • Steppers

    • minuteStep

      Purpose:Number of minutes to step per button press in timebox mode
      Default: 1
      Modes: time, timeflipbox

    • durationSteppers

      Purpose:An object of the amount to step each part of the duration
      Default: {'d':1, 'h':1, 'i':1, 's':1}
      Modes: duration

  • Date Defaults and Limits

    • Defaults

    • defaultPickerValue

      Purpose:Default value shown on the picker when nothing entered into input box. For datebox, slidebox, calbox and flipbox modes array of [y,m,d] is preferred, ISO is accepted. For timebox and timeflipbox modes, array of [h,m,s] is preferred, time value seperated with ":" is also accepted. For durationbox mode, this can be an integer of seconds instead.
      Default: false
      Modes: all

    • defaultDate (deprecated)

      Purpose:This deprecated option is the old name of "defaultPickerValue" option and exists for backward compatibility. Please use "defaultPickerValue" option instead.
      Default: false
      Modes: all

    • Limits

    • rolloverMode

      Purpose:Allow one, or all fields to rollover to next container field - i.e. December + 1 = January of the next year. Individual fields can be set, or a master override of 'false'
      Default:{'m': true, 'd': true, 'h': true, 'm': true, 's': true}
      Modes: all

    • afterToday

      Purpose:Limit date to "today" or after
      Default: false
      Modes: android, calendar

    • beforeToday

      Purpose:Limit date to "today" or before
      Default: false
      Modes: android, calendar

    • notToday

      Purpose:Limit date to NOT "today"
      Default: false
      Modes: calendar

    • maxDays

      Purpose:Set the upper limit to this # of days after today
      Default: false
      Modes: android, calendar

    • minDays

      Purpose:Set the lower limit to this # of days before today
      Default: false
      Modes: android, calendar

    • maxYear

      Purpose:Maximum year allowed in controls
      Default: false
      Modes: android

    • minYear

      Purpose:Minimum year allowed in controls
      Default: false
      Modes: android

    • calWeekMode

      Purpose:Boolean, week mode on or off (choose only a specific DAY)
      Default: false
      Modes: calendar

    • calWeekModeFirstDay

      Purpose:Day of week to choose with week mode (numeric, 0-6, relative to calendar display)
      Default: false
      Modes: calendar

    • blackDays

      Purpose:An array of days of the week to blacklist
      Default: false
      Modes: calendar
      Example: [0,2] : Blacklist sunday and tuesday

    • blackDates

      Purpose:A simple array of individual dates to blacklist
      Default: false
      Modes: calendar
      Example: [ '2010-12-24', '2010-12-25' ] : Blacklist December 24th & 25th, 2010.

    • enableDates

      Purpose:A simple array of individual dates to enable *exclusivly*
      Default: false
      Modes: calendar
      Example: [ '2010-12-24', '2010-12-25' ] : Enable *ONLY* December 24th & 25th, 2010. (note: this option always takes precendence if set)

    • highDays

      Purpose:An array of days of the week to highlight
      Default: false
      Modes: calendar
      Example: [0,2] : Highlight sunday and tuesday

    • highDates

      Purpose:A simple array of individual dates to highlight with pickPageOHighTheme
      Default: false
      Modes: calendar
      Example: [ '2010-12-24', '2010-12-25' ] : Blacklist December 24th & 25th, 2010.

    • useDialogForceTrue

      Purpose:Boolean *Always* use Dialog Window, regardless of screen size

      Modes: all

    • useDialogForceFalse

      Purpose:Boolean *Never* use Dialog Window, regardless of screen size
      Default: false
      Modes: all

    • useModal

      Purpose:Use modal styling (background fades out) on popup
      Default: false
      Modes: all

    • nestedBox

      Purpose:Use when datebox might be initilized somewhere other than where it is opened - i.e. in a simpleDialog
      Default: false
      Modes: all

    • noButtonFocusMode

      Purpose:Suppress button, open on element focus instead
      Default: false
      Modes: all

    • noButton

      Purpose:Suppress button
      Default: false
      Modes: all

    • noSetButton

      Purpose:Suppress the set button in the dialog
      Default: false

    • useInline

      Purpose:Show datebox inline on page, no dialog or popup
      Default: false
      Modes: all

    • useInlineBlind

      Purpose:Show datebox inline on page, rolling it out like window blinds on open
      Default: false
      Modes: all

    • useInlineHideInput

      Purpose:When using inline mode, do not show the input box
      Default: false
      Modes: all

    • openCallback

      Purpose:A function to run on DateBox open (Note: return false from the callback function to prevent datebox opening)
      Default: false
      Modes: all

    • closeCallback

      Purpose:A function to run on DateBox close
      Default: false
      Modes: all

    • swipeEnabled

      Purpose:Swipe gestures active (Note: disableing in flip varients or slide will render them pretty much useless)
      Default: true
      Modes: all
