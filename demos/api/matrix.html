  • Data Formats

    • Output Formats

    • dateFormat

      Type:
      String
      Default:
      Inherited
      Modes:
      all date

      Override the i8n Date format.

      Valid options:

      	YYYY : 4 Digit Year
	MM   : 2 Digit Month (zero pad)
	mm   : 1 or 2 Digit Month
	mmm  : Text Month (monthsOfYear)
	mmmm : Text Month, abbreviated (monthsOfYearShort)
	DD   : 2 Digit Day (zero pad)
	dd   : 1 or 2 Digit Day
	ddd  : Day of week (daysOfWeek)
	dddd : Day of week, abbreviated (daysOfWeekShort)
	SS   : Ordinal Suffix (4th, 3rd, 1st, etc)

    • timeFormats { '12' : String, '24' : String }

      Type:
      Object
      Default:
      See Desc.
      Modes:
      all time

      Override the default time formats. You must supply both a 12 and 24 hour format. i.e. { '12': 'gg:ii AA', '24': 'HH:ii' }

      Valid options:

      	HH : 2 Digit Hour, 24 Hour Clock (zero pad)
	GG : 1 or 2 Digit Hour, 24 Hour Clock
	hh : 2 Digit Hour, 12 Hour Clock (zero pad)
	gg : 1 or 2 Digit Hour, 12 Hour Clock
	ii : 2 Digit Minutes (zero pad)
	ss : 2 Digit Seconds (zero pad)
	aa : Lower case meridiem letters
	AA : Upper case meridiem letters

    • durationFormat

      Type:
      String
      Default:
      'DD ddd, hh:ii:ss'
      Modes:
      all time

      Format the returned duration

      Valid options:

      	DD  : Number of Days (no padding)
	ddd : Text "Day" or "Days" as appropriate from durationDays option
	hh  : 2 or more Digit Hours (zero pad) - can exceed 24 if days not used
	ii  : 2 or more Digit Minutes (zero pad) - can exceed 59 if hours not used
	ss  : 2 or more Digit Seconds (zero pad) - can exceed 59 if minutes not used

    • Input Format

    • timeFormatOverride

      Type:
      Integer
      Default:
      Inherited
      Modes:

    • fieldsOrderOverride

      Type:
      Array
      Default:
      Inherited
      Modes:
      time/date/flip/slide

      Override the i8n fields order. Option is an array of three items, for 24 clock mode 'a' is ignored. i.e. ['m','d','y']

      Valid options:

      	y : Year
	m : Month
	d : Day of Month (date)
	h : Hour
	i : Minute
	a : Meridiem

      Modes:

      Define the order of the fields in the duration

      Valid options:

      	d : Days
	h : Hours
	i : Minutes
	s : Seconds

  • Data Defaults and Limits

    • Defaults

    • defaultPickerValue

      Type:
      Varies
      Default:
      false
      Modes:
      all

      Default value shown on the picker when nothing entered into input box.

      Valid options - dates: An array of [year, month, date] or ISO-8601 date

      Valid options - time: An array of [Hours, Minutes, Seconds] or hours:minutes (24hr)

      Valid options - duration: An integer of seconds
