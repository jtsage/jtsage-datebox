Theming

DateBox is fully jQueryMobile theme-roller compatible (when the theme roller is done!). The individual elements can be passed a theme via the following options:

theme
Original input theme
pickPageTheme
Theme for popup / dialog - controls background colors, and the set button (if present)
pickPageInputTheme
Theme for input elements in the popup / dialog (android, date & duration modes)
pickPageButtonTheme
Buttons (other than the set button) in the popup / dialog
pickPageSlideButtonTheme
Non-selected buttons in slide mode
pickPageTodayButtonTheme
Today's date button in calendar mode
pickPageHighButtonTheme
Currently selected date button in calendar mode
pickPageOHighButtonTheme
Highlighted dates (highDates) in calendar mode
disabledDayColor
Color for text of disabled calendar days - sorry, this isn't a theme yet
