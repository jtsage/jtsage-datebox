Duration Default

Using the duration mode is as simple as setting the 'mode' option to "durationbox".

<label for="mydate">Some Date</label>

<input name="mydate" id="mydate" type="date" data-role="datebox"
   data-options='{"mode": "durationbox"}'>

Duration Short Mode

Using the duration short mode will use only as many digits as nessasary to completly convey the duration.

<label for="mydate">Some Date</label>

<input name="mydate" id="mydate" type="date" data-role="datebox"
   data-options='{"mode": "durationbox", "durationShort": true}'>

Duration No Days Mode

Using the duration no days mode will calculate the duration in h:i:s, never using the # Day(s) prefix.

<label for="mydate">Some Date</label>

<input name="mydate" id="mydate" type="date" data-role="datebox"
   data-options='{"mode": "durationbox", "durationNoDays": true}'>

Configuring Duration Units

Duration units can be defined by setting 'durationOrder' to combinations of ["d","h","i","s"].

<label for="mydate">Some Date</label>

<input name="mydate" id="mydate" type="date" data-role="datebox"
   data-options='{"mode": "durationbox", "durationOrder": ["d", "h", "i"]}'>
