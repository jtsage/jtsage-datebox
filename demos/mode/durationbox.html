A Date and Time Picker plugin for jQueryMobile

Duration Default

Using the duration mode is as simple as setting the 'mode' option to "durationbox".

<label for="mydate">Some Date</label>

<input name="mydate" id="mydate" type="date" data-role="datebox"
   data-options='{"mode": "durationbox"}'>

Duration Collapse Mode

Using the duration collapse mode will use only as many digits as nessasary to completly convey the duration. To use it, set a different durationFormat.

<label for="mydate">Some Date</label>

<input name="mydate" id="mydate" type="date" data-role="datebox"
   data-options='{"mode": "durationbox", "durationFormat": "ii:ss"}'>

Configuring Duration Units

Duration units can be defined by setting 'durationOrder' to combinations of ["d","h","i","s"].

<label for="mydate">Some Date</label>

<input name="mydate" id="mydate" type="date" data-role="datebox"
   data-options='{"mode": "durationbox", "durationOrder": ["d", "h", "i"]}'>
